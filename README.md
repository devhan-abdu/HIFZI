# Hifzi

Hifzi is a Quran companion app that helps users stay consistent with **Hifz** and **Muraja'a**.

The app lets users:

- create structured Hifz plans
- track daily progress and consistency
- follow smart Muraja'a and review schedules
- read the Quran with bookmarks and reading support
- ask Quran-related questions through an AI assistant

## Run the app

```bash
npm run start
```
